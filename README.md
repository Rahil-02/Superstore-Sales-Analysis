# Superstore Sales Analysis

## Project Overview
This project analyzes the Superstore Sales dataset to extract meaningful business insights and support data-driven decision-making.

The analysis was conducted using Google BigQuery for SQL-based querying and Microsoft Power BI for data visualization.

---

## Tools & Technologies
- SQL (Google BigQuery)
- Microsoft Power BI
- Microsoft Excel

---

## Key Objectives
- Analyze sales and profit performance
- Identify top-performing products and categories
- Evaluate regional sales trends
- Understand customer segments and behavior

---

## Key Insights
- Sales are highest in the West region
- Technology category generates the most profit
- Discounted products often reduce profitability
- Corporate segment shows stable purchasing behavior
